# Modelos Conceituais do Estudo

Este repositório disponibiliza versões digitais e ampliáveis dos modelos conceituais desenvolvidos para o estudo sobre a influência dos regimes jurídicos de trabalho no clima organizacional de hospitais universitários federais brasileiros.

Os diagramas foram disponibilizados em formato SVG para permitir visualização em alta resolução, ampliação sem perda de qualidade e melhor compreensão das relações teóricas propostas.

---

## Figura 1 – Modelo Conceitual Integrado: Regimes Jurídicos de Vínculo e Clima Organizacional em Hospitais Universitários Federais Brasileiros

[![Figura 1 – Modelo Conceitual Integrado](figura1_preview.png)](figura1.svg)

*Clique na imagem para visualizar a versão ampliada.*

### Descrição

Este modelo apresenta a estrutura teórica central do estudo, integrando os principais elementos relacionados à coexistência dos regimes jurídicos de trabalho (celetista e estatutário) e seus possíveis reflexos sobre o clima organizacional em hospitais universitários federais brasileiros.

O modelo representa as relações conceituais identificadas na literatura científica, destacando aspectos como:

- Percepção de justiça organizacional;
- Relações interpessoais e interprofissionais;
- Satisfação e motivação no trabalho;
- Gestão de pessoas;
- Impactos institucionais e organizacionais.

### Descrição

Esta figura detalha as variáveis e dimensões analíticas que compõem o modelo conceitual do estudo.

Seu objetivo é explicitar os construtos utilizados para compreender a influência dos diferentes vínculos de trabalho sobre o clima organizacional, permitindo uma visualização integrada das relações entre antecedentes, dimensões intermediárias e possíveis repercussões organizacionais.

A estrutura apresentada serve como referência para a interpretação dos resultados e para futuras investigações relacionadas ao tema.
